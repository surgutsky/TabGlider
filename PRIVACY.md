# Privacy Policy for TabGlider

Last updated: 2026-05-20

## Data Collection

TabGlider does not collect, transmit, or share any user data.

## Data Storage

All data (tab profiles, settings) is stored locally on your device
using Chrome's built-in storage API (chrome.storage.local).
This data never leaves your browser and is not accessible to anyone
other than you.

## Permissions

The permissions requested by TabGlider are used solely to provide
tab profile management functionality:

- tabs: to read and manage browser tabs
- tabGroups: to save and restore tab groups
- storage: to store profile data locally
- sidePanel: to display the profile manager
- scripting: to inject the quick switcher overlay
